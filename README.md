# Infrastructure as Code Project


## SetUp Project Locally 

`````
$ git clone 
$ cd 
`````

## Run/ Test Project 

``````
$ terraform init 
$ terraform plan
$ terraform apply 

$ ansible-vault encrypt secrets.yml
// to deploy the Socks Shop to Cluster 
$ kubectl apply -f socks-shop-deployment.yaml
``````
